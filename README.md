# bluefin-help
Example linux yelp-based help documentation, using `yelp-tools` for freedesktop.org compliant linux help.

## Dev Container
Open this project using the given devcontainer specification

## Building

```bash
make
```

Output file will be `index.epub`

## Cleaning

```bash
make clean
```

## Reading Help File

From the host:

```bash
yelp .
```
